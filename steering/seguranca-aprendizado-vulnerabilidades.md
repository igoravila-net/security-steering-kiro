# Aprendizado Contínuo - Vulnerabilidades Detectadas

> Este arquivo é atualizado automaticamente pelo hook `learn-from-vulnerabilities` sempre que o hook de revisão de segurança bloqueia código vulnerável. Os dados coletados são usados para melhorar os steerings existentes e identificar gaps.

## Como Funciona

1. Desenvolvedor tenta escrever código com vulnerabilidade
2. Hook `security-code-review` bloqueia a escrita
3. Hook `learn-from-vulnerabilities` registra o padrão aqui
4. Periodicamente, os dados são analisados para:
   - Identificar vulnerabilidades mais frequentes
   - Melhorar steerings existentes com novos exemplos
   - Criar novos steerings para gaps identificados
   - Ajustar regras que geram muitos falsos positivos
